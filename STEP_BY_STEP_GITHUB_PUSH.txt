📋 COMPLETE STEP-BY-STEP PROCESS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: CREATE GITHUB REPOSITORY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Open your browser and go to: https://github.com/new

2. Fill in the repository details:
   - Repository name: cognitive-fatigue-detection
   - Description: Advanced AI-powered cognitive fatigue detection system with 
                  MongoDB
   - Choose: Public (anyone can see) or Private (only you can see)
   - DO NOT check "Initialize this repository with:"
   - DO NOT add .gitignore or license
   - Click "Create repository"

3. After creation, you'll see a page with instructions
   You'll need your repository URL which looks like:
   https://github.com/YOUR_USERNAME/cognitive-fatigue-detection.git

   ✨ SAVE THIS URL - YOU'LL NEED IT!


STEP 2: VERIFY GIT IS INSTALLED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Open PowerShell and run:

    git --version

Expected output: git version 2.x.x (or higher)

If you see "git: command not found", download from: https://git-scm.com/download/win


STEP 3: OPEN PROJECT FOLDER IN POWERSHELL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run this command:

    cd "c:\PFSD PROJECT\cognitive-fatigue-detection"

✓ You should see the folder path in your terminal


STEP 4: CHECK GIT STATUS (VERIFY ALL FILES ARE COMMITTED)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run:

    git status

Expected output:
    On branch master
    nothing to commit, working tree clean

If you see "nothing to commit" → Good! All files are ready.
If you see untracked files → Run: git add . && git commit -m "final commit"


STEP 5: ADD GITHUB REPOSITORY AS REMOTE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Replace YOUR_USERNAME with your actual GitHub username in the URL below:

    git remote add origin https://github.com/YOUR_USERNAME/cognitive-fatigue-detection.git

Example (if your username is "john_dev"):

    git remote add origin https://github.com/john_dev/cognitive-fatigue-detection.git

✓ This command tells git where your GitHub repository is


STEP 6: VERIFY REMOTE CONNECTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run:

    git remote -v

Expected output:
    origin  https://github.com/YOUR_USERNAME/cognitive-fatigue-detection.git (fetch)
    origin  https://github.com/YOUR_USERNAME/cognitive-fatigue-detection.git (push)

✓ If you see this, remote is properly connected!


STEP 7: RENAME BRANCH FROM 'master' TO 'main' (Optional but Recommended)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run:

    git branch -M main

This renames your local branch to 'main' (GitHub's default)


STEP 8: PUSH YOUR CODE TO GITHUB - METHOD A: HTTPS (RECOMMENDED)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run:

    git push -u origin main

When prompted:
  Username: Enter your GitHub username
  Password: Enter your GitHub Personal Access Token (NOT your password!)

Note: See section "GET GITHUB PERSONAL ACCESS TOKEN" below


STEP 8B: PUSH YOUR CODE TO GITHUB - METHOD B: SSH (More Secure)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

If you have SSH keys set up:

1. First, change the remote URL:

    git remote set-url origin git@github.com:YOUR_USERNAME/cognitive-fatigue-detection.git

2. Then push:

    git push -u origin main

No password needed - SSH uses key authentication

⚠️  GET GITHUB PERSONAL ACCESS TOKEN (If Using HTTPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GitHub no longer allows plain passwords for HTTPS pushes. You need a Personal
Access Token (PAT). Here's how to get one:

1. Go to: https://github.com/settings/tokens

2. Click "Generate new token" (classic)

3. Fill in the form:
   - Note: "Cognitive Fatigue Detection Push"
   - Expiration: Choose 90 days (or longer)
   - Select scopes (checkboxes):
     ✓ repo (Full control of private repositories)
     ✓ write:repo_hook
     ✓ delete:repo (optional)

4. Scroll down and click "Generate token"

5. COPY THE TOKEN IMMEDIATELY (you won't see it again!)
   Save it somewhere safe like:
   - Password manager
   - Secure notes
   - Text file (NOT in git!)

6. Use this token as your "password" when git prompts you

❓ TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLEM: "fatal: remote origin already exists"
SOLUTION: 
    git remote remove origin
    Then run the "git remote add origin" command again

───────────────────────────────────────────────────────

PROBLEM: "Authentication failed" or "Invalid username or password"
SOLUTION:
    Make sure you're using a Personal Access Token, NOT your GitHub password
    Get a new token from: https://github.com/settings/tokens

───────────────────────────────────────────────────────

PROBLEM: "Could not resolve host: github.com"
SOLUTION:
    Check your internet connection
    Try: ping github.com
    If offline, connect to internet and try again

───────────────────────────────────────────────────────

PROBLEM: "refused to merge unrelated histories"
SOLUTION:
    Run: git pull origin main --allow-unrelated-histories
    Then: git push -u origin main

───────────────────────────────────────────────────────

PROBLEM: "fatal: 'origin' does not appear to be a 'git' repository"
SOLUTION:
    Make sure you're in the correct folder:
    cd "c:\PFSD PROJECT\cognitive-fatigue-detection"
    Then run: git remote add origin [your-url]

───────────────────────────────────────────────────────

PROBLEM: "Changes rejected" or "permission denied"
SOLUTION:
    You might not have write access to the repository
    Make sure you own the GitHub repository
    Or the repository owner needs to add you as a collaborator

───────────────────────────────────────────────────────

PROBLEM: "fatal: could not read Username"
SOLUTION (Save credentials):
    git config --global credential.helper store
    Then run: git push -u origin main
    Enter your credentials once, they'll be saved for future pushes

───────────────────────────────────────────────────────

PROBLEM: "Repository does not exist or you don't have permission"
SOLUTION:
    Verify your repository URL is correct
    Check the repository exists at: https://github.com/YOUR_USERNAME/cognitive-fatigue-detection
